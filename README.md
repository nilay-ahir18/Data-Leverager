# DataLeverager Power BI ETL Project

## Project Overview

This project demonstrates the complete ETL (Extract, Transform, Load) process using Microsoft Power BI Power Query.

The project integrates data from multiple Excel files and a web source, performs data cleaning, transformation, aggregation, and refresh simulation.

---

## Data Sources

### Web Source

* GDP data imported from Wikipedia

### Excel Sources

* Sales_Jan.xlsx
* Sales_Feb.xlsx
* Sales_Mar.xlsx
  
* Employees.xlsx

---

## Transformations Performed

### Data Cleaning

* Removed blank rows
* Removed duplicates
* Changed data types
* Renamed columns

### Text Transformations

* Trim
* Clean
* Uppercase
* Lowercase
* Split Column

### Numeric Transformations

* Revenue Rounding
* Profit Calculation

### Date Transformations

* Year Extraction
* Month Extraction
* Quarter Extraction
* Day Extraction
* Employee Age Calculation

### Conditional Logic

* Sales Category Creation

### Data Integration

* Merge Queries
* Append Queries

### Data Reshaping

* Pivot Columns
* Unpivot Columns

### Aggregation

* Total Sales
* Average Sales
* Transaction Count

### Data Profiling

* Column Quality
* Column Distribution
* Column Profile

### Parameterization

* Dynamic Folder Path Parameter
